# Hardware Notes

## Кондиционер
- Модель: Mystery MSS-09R07M
- Мощность: 9000 BTU
- Тип: мобильный

## Компоненты

### ESP32-CAM
- Контроллер + Wi-Fi + Bluetooth
- Встроенная камера (опционально: читать дисплей кондея визуально)
- GPIO для управления PC817
- Питание: 5V USB

![ESP32-CAM](photos/esp32-cam.jpg)

### 8-канальный модуль PC817 (оптоизоляция)
- 8 каналов — перекрывает все кнопки панели кондея
- Гальваническая развязка между ESP32 и схемой кондея
- Входное напряжение управления: 3.3V (совместимо с ESP32)
- Преимущество перед BC547: не нужны отдельные резисторы, физическая изоляция

![PC817 module](photos/pc817-module.jpg)

### DHT22
- Датчик температуры и влажности
- Точность: ±0.5°C, ±2-5% RH
- Протокол: 1-Wire

## Принцип работы
Каждая кнопка панели кондея подключается параллельно через оптопару PC817:
- ESP32 GPIO → резистор 1кОм → анод PC817
- Катод PC817 → GND
- Коллектор/Эмиттер PC817 → параллельно кнопке кондея

При подаче HIGH на GPIO — оптопара замыкает кнопку.
