# Hardware Notes

## Кондиционер
- Модель: Mystery MSS-09R07M
- Мощность: 9000 BTU
- Тип: мобильный

## Компоненты

### ESP32-CAM
- Контроллер + Wi-Fi + Bluetooth
- Встроенная камера (опционально: читать дисплей кондея визуально)
- GPIO для управления PC817
- Питание: 5V USB

### 8-канальный модуль PC817 (оптоизоляция)
- 8 каналов — перекрывает все кнопки панели кондея
- Гальваническая развязка между ESP32 и схемой кондея
- Входное напряжение управления: 3.3V (совместимо с ESP32)

### DHT22
- Датчик температуры и влажности
- Точность: ±0.5°C, ±2-5% RH
- Протокол: 1-Wire

## Ключевой чип: 74HC164 (сдвиговый регистр)

**Важно!** Светодиоды и дисплей управляются **НЕ прямыми пинами**, а через сдвиговый регистр 74HC164.

- Тип: 8-бит Serial-In, Parallel-Out (SIPO)
- Корпус: 14-pin SOIC (на обратной стороне платы)

**Это означает:**
- Главная плата шлёт последовательный поток: DATA + CLK
- 8 тактов CLK = 1 байт = состояние 8 LED/сегментов
- Возможно 2 чипа 74HC164 в цепочке (16 бит = 10 LED + сегменты дисплея)

**ESP32 подключается параллельно** к DATA и CLK — только читает, ничего не пишет.

```
Главная плата кондея
       ↓ DATA + CLK
   74HC164 #1 → 8 бит → LED (AUTO, COOL, FAN, HEAT, HI, LOW, COMP, SLEEP)
   74HC164 #2 → 8 бит → LED (TIME, W/F) + сегменты дисплея
                  ↑
     ESP32 слушает здесь (параллельно)
```

## Разъём CN1 (8 пинов)

Соединяет плату управления с главной платой кондея.

**Предполагаемая распиновка (нужно прозвонить!):**

| Пин | Вероятная функция |
|-----|------------------|
| 1 | VCC (+5В) |
| 2 | GND |
| 3 | DATA (к 74HC164) |
| 4 | CLK (к 74HC164) |
| 5-7 | Кнопки (матрица или прямые линии) |
| 8 | Общий / доп. сигнал |

---

## Схема подключения ESP32

```
Кнопки (управление):
ESP32 GPIO → резистор 1кОм → PC817 анод
PC817 катод → GND
PC817 коллектор/эмиттер → параллельно кнопке кондея
HIGH на GPIO = нажатие кнопки

Светодиоды (чтение):
CN1 DATA → ESP32 GPIO (вход, только чтение)
CN1 CLK  → ESP32 GPIO (вход, только чтение)
ESP32 перехватывает биты и декодирует состояние LED
```
